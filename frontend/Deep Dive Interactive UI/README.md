# Deep Dive Into Interactive UI — JS Core & DOM

This is a compact interactive playground designed to teach and demonstrate core browser UI concepts using plain JavaScript and DOM APIs.

Features
- Programmatic tabs and panel switching
- Modal with focus handling
- Todo list demonstrating state persistence (localStorage)
- Drag & drop reordering
- DOM inspector (live element info, highlight)
- Keyboard shortcut demo

Learning goals
- Understand DOM selection, event handling, and event delegation
- Learn to manipulate element attributes and properties
- Manage simple state and persistence (localStorage)
- Implement accessible UI patterns (tabs, modal basics)

Run
1. Open `index.html` in a browser (no build step required).
2. Try the tabs, add todos, drag list items, or toggle inspecting mode.
